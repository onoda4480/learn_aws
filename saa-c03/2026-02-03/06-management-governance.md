# マネジメント・ガバナンス

## AWS Migration Hub

オンプレからAWS移行する際の進捗を管理するダッシュボード

### 役割
```
実際に移行する ✗
移行の進捗を管理する ⭕
```

### 関連ツール

| ツール | 役割 |
|-------|------|
| Migration Hub | 全体管理・ダッシュボード |
| AWS MGN | サーバーを移行 |
| AWS DMS | データベースを移行 |
| App2Container | アプリをコンテナ化 |

## AWS Control Tower / Account Factory

### Control Tower
複数AWSアカウントの統制・管理サービス

### Account Factory
Control Tower の中の「アカウント量産機能」

- テンプレートで標準化されたAWSアカウントを自動作成
- VPC構成、リージョン、SSO設定、ガードレールを統一
- 手作業のミスを防ぎ、何個作っても同じ品質
- 企業のガバナンス維持に必須（個人利用では不要）

## AWS Step Functions

複数の処理を順番に繋げて自動化するサービス

### 2つのワークフロータイプ

| 項目 | Standard | Express |
|-----|----------|---------|
| 実行時間 | 最大1年 | 最大5分 |
| 実行回数 | 少なめ向き | 大量処理向き |
| 料金 | 状態遷移ごとに課金 | 実行回数+時間で課金 |
| 実行保証 | 1回だけ実行 | 最低1回実行（重複あり） |

### Express ワークフロー
大量・高速・短時間の処理に特化

#### 2つのモード
- **同期**：結果を待つ（API Gateway → Express → レスポンス）
- **非同期**：結果を待たない（大量メッセージを次々処理）

### 使い分け
```
5分以内で終わる？    → Express
それ以上かかる？     → Standard
人の承認待ちがある？ → Standard
大量リクエスト処理？ → Express
```
