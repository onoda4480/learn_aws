# ストレージ選択（試験問題）

## 問題パターン：用途別ストレージ選択

### シナリオ
- 動画編集アプリ（複数EC2で共有アクセス）
- 編集中は高I/Oアクセスが必要
- 編集後はアーカイブ保存

### 正解の組み合わせ

| 用途 | 最適なストレージ | 理由 |
|------|-----------------|------|
| 共有ファイル | **EFS** | 複数EC2から同時アクセス可能 |
| 高速I/O処理 | **Instance Store** | 物理的に接続、最高のI/O性能 |
| 長期アーカイブ | **S3** | 低コスト、耐久性99.999999999% |

## Instance Store vs EBS

| 項目 | Instance Store | EBS |
|-----|----------------|-----|
| 接続方式 | 物理的に接続 | ネットワーク経由 |
| I/O性能 | **最速** | 高速（io2で高IOPS可能） |
| データ永続性 | なし（停止で消える） | あり |
| 用途 | キャッシュ、一時データ | OS、DB、永続データ |

## 試験キーワード

```
「最高のI/Oパフォーマンス」「一時的な処理」→ Instance Store
「複数EC2から共有」→ EFS
「長期保存」「アーカイブ」→ S3
```
