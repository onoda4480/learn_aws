# セキュリティ

## AWS CloudHSM

暗号鍵を保管・管理する**専用ハードウェア**

### KMS との違い

| 項目 | CloudHSM | KMS |
|------|----------|-----|
| 管理 | **自分で管理** | AWSが管理 |
| ハードウェア | **専用（占有）** | 共有 |
| 鍵の管理権限 | **完全に自分** | AWSと共有 |
| コンプライアンス | FIPS 140-2 Level 3 | FIPS 140-2 Level 2 |
| 料金 | 高い | 安い |

### イメージ

```
KMS：AWSの金庫に鍵を預ける（AWSも鍵にアクセス可能）
CloudHSM：自分専用の金庫を借りる（AWSは鍵に触れない）
```

### 試験ポイント

```
「専用ハードウェア」「完全な管理権限」「FIPS 140-2 Level 3」→ CloudHSM
「マネージド」「簡単に暗号化」→ KMS
```
